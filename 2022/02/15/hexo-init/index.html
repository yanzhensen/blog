Hexo 是快速、简洁且高效的博客框架

Hexo官网:https://hexo.io/zh-cn/

Hexo搭建步骤

1.安装Git
2.安装Node.js
3.安装Hexo
4.Gitee创建个人仓库

具体实现

1.GIT

Git下载地址:https://git-scm.com/downloads
Git安装教程 :https://blog.csdn.net/sishen47k/article/details/80211002

2.Node.js

Node.js下载地址:https://nodejs.org/en/download/
Node.js安装教程:https://www.runoob.com/nodejs/nodejs-install-setup.html

2.Hexo

  • 所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo。

    npm install -g hexo-cli

  • 安装hexo-deployer-git，跟git交互

    npm install hexo-deployer-git –save

  • 创建一个文件夹hexo

    mkdir hexo

  • 进入hexo

    cd hexo

  • 初始化一下hexo

    hexo init

  • 安装依赖

    npm install

  • 启动服务

    hexo serve

  • 部署生成页面

    hexo g

  • 清除已生成的页面

    hexo clean

  • 推送远端git

    hexo d

  • 开始部署博客并推到远端git 组合指令

    hexo g -d

  • 强制部署

    hexo d -fg

    d：构建静态文件。
    -f：强制部署，即使当前没有任何更新，也会重新部署所有文件。
    -g：在部署之前强制生成所有静态文件

  • 文件夹出现的名称

文件名称 描述
node_modules 依赖包
public 存放生成的页面
scaffolds 生成文章的一些模板
source 用来存放你的文章
themes 主题
_config.yml 博客的配置文件

主题

  • 模板使用：https://gitee.com/mirrors_blinkfox/hexo-theme-matery.git
  • 安装到themes目录下

    cd themes
    git clone https://gitee.com/mirrors_blinkfox/hexo-theme-matery.git

  • 修改 Hexo 根目录下的 _config.yml 的 theme 的值：theme: hexo-theme-matery

安装完后即可用hexo s在本地启动，然后访问localhost:4000查看你的博客了

安装hexo-wordcount字数统计插件

  1. 在根目录安装hexo-wordcount
npm i --save hexo-wordcount
  1. 在自己主题中的修改_config.yml
    我这里是matery主题
cd themes/hexo-theme-matery
vim _conf.yml

修改_conf.yml

# Post word count, reading duration, site total word count.
# Before you activate, please confirm that you have installed the hexo-wordcount plugin,
# install the plugin command: `npm i --save hexo-wordcount`.
# 文章字数统计、阅读时长、总字数统计等
# 文章信息--若要开启文章字数统计，需要安装 hexo-wordcount 插件，安装命令: `npm i --save hexo-wordcount`
postInfo:
  date: true # 发布日期
  update: true # 更新日期
  wordCount: true # 文章字数统计
  totalCount: true # 站点总文章字数
  min2read: true # 文章阅读时长
  readCount: true # 文章阅读次数

引入live2D 动漫人物

  1. 在根目录安装live2D依赖

模板有很多，请自行去查找 这里引用的是koharu

#引入live2d依赖
npm install --save hexo-helper-live2d
#引入koharu模型 npm install {packagename}
npm install live2d-widget-model-koharu
  1. 在根目录下配置 _conf.yml 文件
# live2D
live2d:
  enable: true
  scriptFrom: local
  model:
    use: live2d-widget-model-koharu #模型选择
  display:
    position: left   #模型位置
    width: 150       #模型宽度
    height: 300      #模型高度
  mobile:
    show: false      #是否在手机端显示

大功告成啦！！
